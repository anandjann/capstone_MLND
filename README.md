# Machine-learning-nanodegree-capstone
The capstone project for Udacity Machine learning nanodegree.
problem: Classify iceberg or ship.

2. Python : 3.6
3. Anaconda 64 bit v4.3 distribution.

2. Pandas

4. seaborn
5. keras
